Concepts:
- **Opening Book**: Database of known opening positions with popular moves, weights, statistics
- **Tablebase**: Perfect endgame solutions for positions with few pieces (DTM = Distance To Mate)
- **Repertoire**: Named opening system/style (e.g., "aggro_mainline", "solid_defense")

Data Sources:
- Opening books: Polyglot format or custom JSON
- Tablebases: Syzygy format (3-7 piece endgames)
