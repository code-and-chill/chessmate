## ‚ú® Key Concepts

### PlayScreenConfig
Unified configuration combining board, theme, and API settings. Fully typed and extensible.

### BoardConfig
Board presentation configuration (size, square size, border radius, interactivity, disabled opacity).

### ThemeConfig
Theme configuration (light/dark mode, board theme color, custom colors).

### useGameParticipant Hook
Validates game participation and assigns player colors.

### useGameInteractivity Hook
Determines board interactivity state with reason codes.

### Configuration Factories
Functions that create configurations based on parameters:
- `createPlayScreenConfig()` - Merge custom config with defaults
- `createResponsiveBoardConfig()` - Create responsive board sizing

## üöÄ Quick Start

```typescript
// Basic usage (no changes needed)
<PlayScreen gameId="game-123" />

// Dark theme with blue board
<PlayScreen
  gameId="game-123"
  config={{
    theme: { mode: 'dark', boardTheme: 'blue' }
  }}
/>

// Responsive board on mobile
const boardConfig = createResponsiveBoardConfig(screenWidth);
<PlayScreen gameId="game-123" config={{ board: boardConfig }} />
```
