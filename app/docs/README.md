# Chess App

Mobile (iOS/Android) and web user interface for Chessmate platform.

## Overview

Chess App is a cross-platform application built with React Native and React providing:
- Mobile UI (iOS/Android via React Native)
- Web UI (Desktop/Tablet via React)
- Player authentication and profile management
- Live game interface
- Game lobby and browsing
